# tempconv
Simple temperature convert. Converts between 째C, 째F and K.


### Installation
``npm i tempconv``


### Simple to use
```js
const { Convert } = require('tempconv');
console.log(Convert('F', 'K', 1)); // Converts 1 Fahrenheit to Kelvin.
```


### Valid usage
**Shortened unit**

째C, C, 째F, F and K.


**Full unit**

Celsius, Fahrenheit and Kelvin.
